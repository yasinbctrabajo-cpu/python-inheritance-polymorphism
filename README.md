# Herencia y Polimorfismo en Python

## Descripción

Repositorio que contiene ejercicios orientados al uso de herencia y polimorfismo dentro de la programación orientada a objetos.

Estos conceptos permiten crear aplicaciones más flexibles, reutilizables y fáciles de mantener.

## Tecnologías utilizadas

- Python 3
- Jupyter Notebook

## Conceptos trabajados

- Herencia
- Polimorfismo
- Sobrescritura de métodos
- Reutilización de código
- Jerarquías de clases
- Programación orientada a objetos avanzada

## Objetivo

Comprender cómo extender y reutilizar funcionalidades mediante herencia y polimorfismo para desarrollar aplicaciones más robustas.
